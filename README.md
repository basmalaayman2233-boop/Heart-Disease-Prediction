# Heart Disease Prediction Using Machine Learning

This project aims to predict the presence of heart disease using multiple machine learning models and compare their performance.

##  Dataset
- Heart Disease Dataset
- Features include age, sex, cholesterol level, blood pressure, and more.

##  Machine Learning Models Used
- Decision Tree Classifier
- Logistic Regression
- Random Forest Classifier
- Support Vector Classifier (SVC)
- K-Means Clustering
- Hierarchical Clustering

##  Steps
1. Data preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature scaling
4. Training multiple ML models
5. Model evaluation and comparison

##  Evaluation Metrics
- Accuracy
- Confusion Matrix
- Classification Report

##  Technologies
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
